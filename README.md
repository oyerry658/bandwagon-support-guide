## 搬瓦工客服的本质：工单制，没有实时响应

搬瓦工（BandwagonHost）的客服支持，本质上是一个工单系统（Ticket System）。你在后台提交问题，等客服看到后回复到你邮箱，然后你再在工单里继续跟进。

不是实时的。没有弹窗客服。没有电话。

这种模式在国外的 VPS 行业里很普遍，搬瓦工卖的是 Unmanaged VPS（无管理型），意思是他们负责硬件和网络，你自己管服务器里的事。所以大部分使用问题，按这个定位，其实轮不到客服管。

但属于他们范围内的问题——网络大规模丢包、服务器宕机、账单异常、退款申请——找他们是没问题的，也确实会处理。

## 提交工单的完整步骤

### 方法一：后台登录提交（推荐）

1. 登录搬瓦工账号后台
2. 点击顶部菜单 **Support → Tickets**
3. 点击 **Open New Ticket**（新建工单）
4. 填写 Subject（主题）、Related Service（关联的 VPS，下拉选）、Message（问题描述）
5. 如果有截图，点击选择文件上传
6. 点击 **Submit** 提交

### 方法二：无法登录账号时

直接发邮件到搬瓦工官方支持邮箱，无需登录后台也能提交。这个方式适合账号密码忘了、或者邮箱验证出问题导致登不进去的情况。

## 必须用英文吗？

严格来说，是的。

工单页面默认英文环境，客服团队也是英语沟通。自己用中文写好问题，再扔进翻译工具（比如 DeepL 或者百度翻译）翻成英文，粘贴过去就行了，不需要多准确，能让老外看懂大概意思就够了。

我自己也没有专门写过特别标准的英文工单。语法不顺也没关系，只要把问题说清楚——什么时候开始、什么现象、你自己做过什么排查——客服基本都能处理。

## 工单多久会回复？几个必须知道的时区坑

这是很多人踩的坑。

搬瓦工在加拿大温哥华，比北京时间慢 16 小时。工作日正常的情况下，工单一般能在 24 小时内有回复，快的话几小时。

但有两个时间点要注意：

**周末不处理工单。** 周六、周日提交的工单，要等到周一才会有人看。这个等下来可能要两三天。如果你的问题不紧急，就耐心等；如果 VPS 完全挂了，先去 bwhstatus.com 看看是不是搬瓦工那边的机房问题，是的话他们会自己修，不用你催。

**重复回复会适得其反。** 工单系统里，越新的工单/回复排在越后面。你催一次，工单的优先级反而往后排。提交完了就等着，别连续回复"有没有人""在吗"之类的，帮倒忙。

## 出现 Martin C 机器人回复怎么办

这个梗在搬瓦工用户里流传很广。

提交工单后，有时会在半小时内收到一个叫 **Martin C** 的回复，内容通常是自动化的、答非所问的格式化文字。多年经验来看，这基本就是个自动回复机器人，不是真人客服。

碰到这种情况，不用慌。把你的问题原文复制一遍，重新回复在工单里，下次就会有真人来处理。

## 什么问题找客服有用，什么问题找了也没用

这点很多新手搞不清楚，结果提了工单发现客服根本不管，白等了两天。

**客服会处理的：**
- 网络大面积异常（全球都 ping 不通、丢包率极高）
- 服务器宕机且确实是机房那边的问题
- 账单问题：误扣费、升级降级、续费相关
- 退款申请（30 天内可申请全额退款）
- IP 被封需要换 IP（有时需要工单操作）
- KiwiVM 后台权限或访问问题

**客服不管的（别浪费时间）：**
- 怎么在服务器上安装 XXX 软件
- 代码跑不起来
- VPN 配置方法
- 网站打不开但 VPS 还活着
- 操作系统配置、防火墙、SSH 连不上（这些都是你自己的事）

搬瓦工卖的是"裸机"，你租了地方，里面装什么、怎么用，都自己负责。这本来就是 Unmanaged VPS 的游戏规则。

## 常见问题 FAQ

**Q：搬瓦工有没有中文客服？**

没有。搬瓦工是加拿大公司，客服全英文，没有中文坐席，也没有中文在线聊天。联系方式只有后台工单和官方邮件。

**Q：工单可以用中文写吗？**

严格说不行，应该用英文。自己把中文问题翻译一下再提交，翻译软件就够用了，不用写标准英语。

**Q：搬瓦工客服什么时候有人？**

工作日（周一到周五）都有人处理工单，一般 24 小时内回复。周六周日不工作，周末提的工单要等到周一才处理。时区是加拿大温哥华时间，比北京慢 16 小时。

**Q：收到 Martin C 的回复是怎么回事？**

Martin C 是搬瓦工的自动回复机器人，不是真人。收到之后，把问题原文再回复一遍到工单里，会触发真人客服处理。

**Q：搬瓦工退款好不好退？**

购买后 30 天内，全额退款，走工单申请。实测可以退，填好退款原因、订单号，等回复就行。过了 30 天就不退了。

**Q：哪些问题不要找客服？**

软件安装、代码报错、VPN 配置、建站教程——这些搬瓦工不管，工单提了也是白等。这类问题自己查或者找论坛。
